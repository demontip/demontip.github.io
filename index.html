用C++设计一个不能被继承的类

发表于 2019-04-09 | 分类于 C++ | 阅读次数:

方法一. 定义静态函数

由于子类会调用父类的构造函数，一个类要想不能被继承，可以把它的构造函数和析构函数定义为私有函数，这样子类构造时就会报错。

但同时由于构造函数是私有的，也就无法创建实例了。这时需要再定义静态函数来创建和释放实例。

在这里，子类不可以继承A，同时可以通过A b = A::GetA() 来定义一个A的实例。（**因为成员函数GetA()是static的，所以可以通过类名直接调用。***）

方法二. 友元和模板

方法一只能局限于在堆上创建实例，并且通过这种额外定义函数的方式，感觉破坏了类的原有形式。更好的解决方式如下：

让A类再虚拟继承一个父类Afinal，这个父类的构造和析构函数是私有的，并且将A类声明为友元。那A类在创建实例的时候，就可以访问这个父类Afinal的构造和析构函数了，同时A类不允许继承，因为继承的子类会跳过A类，直接调用Ainal的构造函数，显然这样会发生错误，因为Afinal的构造函数是私有的。

发表于 2018-11-27 | 分类于 LeetCode题目详解 | 阅读次数:

题目描述

Implement atoi which converts a string to an integer.

The function first discards as many whitespace characters as necessary until the first non-whitespace character is found. Then, starting from this character, takes an optional initial plus or minus sign followed by as many numerical digits as possible, and interprets them as a numerical value.
The string can contain additional characters after those that form the integral number, which are ignored and have no effect on the behavior of this function.
If the first sequence of non-whitespace characters in str is not a valid integral number, or if no such sequence exists because either str is empty or it contains only whitespace characters, no conversion is performed.
If no valid conversion could be performed, a zero value is returned.

Note:

  • Only the space character ' ' is considered as whitespace character.
  • Assume we are dealing with an environment which could only store integers within the 32-bit signed integer range:$[-2^{31}, 2^{31}-1]$.If the numerical value is out of the ranges of representable values, INT_MAX $(2^{31}-1)$ or INT_MIN $-(2^{31})$ is returned.

难度： Medium

题目大意

​ 输入一串字符串，要求从第一个非空字符（空字符即' '）检查，如果检查出如下结果，则表明成功检查到整数： 第一个字符为 ‘+’或者'-'或者无符号，且后面跟着至少一个'0'~'9'的数字字符，（不包含小数点） 。若检查成功，输出该整数的32位int型表示，检查失败，输出0。如果检查到的数超出了32位int的范围：$[-2^{31}, 2^{31}-1]$，则相应的输出最大值INT_MAX $2^{31}-1$ , 或者最小值INT_MIN $-(2^{31})$。
