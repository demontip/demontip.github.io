SpringBoot - 利用JPA+MySQL实现数据库连接、多表查询

发表于 2019-05-03 | 分类于 Spring | 阅读次数:
字数统计: 2.4k

环境

win10
jdk1.8
maven-3.6.1
sts-3.9.8
MySQL 8.0

目的

在这个例子中，我将建立一个SpringBoot项目，并用JPA来连接MySQL数据库。
数据库中我会建立两张表Customer1和Customer2，Customer1有id、firstName、lastName三个属性，Customer2有id、age、sex三个属性。两张表中若id相同，则表示同一个人。
用JPA连接Customer1/2，实现在每张表中按条件查找，以及查找两张表中id相同的人并将这个人的所有属性（firstName、lastName、age、sex）整合在一起输出。

2.引入JPA、JDBC、MySQL、WEB依赖，点击完成。

3.新建好项目后，依次添加各个类，方便管理，全部的文件如下图所示。

这里对几个新建的类先大概介绍下，后面会给出每个类的具体代码，github代码地址。
com.example.demo
​ 存放的是新建项目时自动生成的Application启动类，无需修改。
com.example.demo.bean
​ Customer1.java ：定义Customer1表对应的的实体类
​ Customer2.java ：定义Customer2表对应的的实体类
​ Params.java：定义Customer1与Customer2整合后的输出类
com.example.demo.dao
​ Customer1Repository.java：定义操作Customer1表需要的函数的接口类
​ Customer2Repository.java：定义操作Customer2表需要的函数的接口类
com.example.demo.service
​ Customer1Service.java：定义在Customer1上进行增删查等操作的服务，用于将Repository封装起来
​ Customer2Service.java：定义在Customer2上进行增删查等操作的服务，用于将Repository封装起来
com.example.demo.controller:
​ Customer1Controller.java：定义用于web连接时的函数接口
​ Customer2Controller.java：定义用于web连接时的函数接口
